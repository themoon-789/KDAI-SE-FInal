# Advance Agent for Cybersecurity - Demo

ระบบจัดการความมั่นคงปลอดภัยทางไซเบอร์สำหรับองค์กร

## คุณสมบัติหลัก

1. **Knowledge Management** - จัดการเอกสารและองค์ความรู้เกี่ยวกับภัยคุกคาม
2. **Log Management** - รวบรวมและจัดการ Syslog จากอุปกรณ์ต่างๆ
3. **System Monitoring** - Dashboard แสดงสถานะระบบแบบ Real-time

## การติดตั้ง

```bash
cd cybersecurity_demo
pip install -r requirements.txt
python app.py
```

เปิดเบราว์เซอร์ที่ http://localhost:5000
